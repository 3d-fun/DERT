Configuration

  • Open a landscape: Select File>Open Landscape ... and browse to a landscape. Select the landscape and then Choose.
  • Open an existing configuration: Select File>Open Landscape ... and browse to a landscape. Select a landscape, the configuration from the list at the bottom of the dialog, and then OK.
  • Remove a configuration: Select File>Delete Configuration.
  • Save the current configuration: Select File>Save Configuration or File>Save Configuration As.

Viewpoint
  • Translate the viewpoint along a plane parallel with the terrain: Place the cursor over the terrain, press the left mouse button, and drag. The terrain movement will die off gradually after the mouse release. Press any mouse button to stop the movement immediately.
  • Translate the viewpoint along the plane of the screen: Place the cursor over the terrain, press the middle mouse button, and drag.
  • Rotate the viewpoint about the center of the view: Place the cursor over the terrain, press the right mouse button, and drag.
  • Dolly the viewpoint in and out from the center of the view: Move the mouse Scroll wheel.
  • Set the center of rotation: Translate the viewpoint along the plane of the screen until the cross-hair is in the desired location. Alternately, single-click with the right mouse button at the desired location and select Set Center of Rotation from the menu.
  • Change magnification (zoom) of the view: Press the magnify mode button. Use the mouse scroll wheel to change the magnification level. Press the magnify button again to go back to dolly mode.
  • Set the viewpoint to the home position: Press the home button (overhead view, center of rotation at landscape center, magnification = 1, rotation = identity).
  • Add the current viewpoint to the list of saved viewpoints: Open the viewpoint view and press Add.
  • Remove a viewpoint from the list of saved viewpoints: Open the viewpoint view and select the viewpoint from the list. Press Delete.
  • Move to the previous viewpoint in the saved list: Open the viewpoint view and press the left arrow.
  • Move to the next viewpoint in the saved list: Open the viewpoint view and press the right arrow.

Marble
  • Place the green marble somewhere in the scene: Single-click with the left mouse button at the desired location. Alternatively, type the coordinates in the marble location field at the top right of the worldview and press return.
  • Open the marble info view: Press the marble button.
  • Find the coordinates at a given point: Single-click at the desired location.  See the coordinates in the marble location field. Additional information can be found in the marble view.

Map Elements
  • Add a map element (Placemark, Figure, Billboard, Path, Plane, Grid, Camera, Profile) to the scene: Right-click and select Add>map element type from the context menu. You may also open the Map Elements view, select the map element category, and press the appropriate Add button. To add a lineset, open the Map Elements view and select the LineSets category, then press the Add button.
  • Delete a map element: Right-click on the element and select Delete from the context menu. The element may also be deleted from the Map Elements view.
  • Hide a map element: Right-click on the element and select Hide from the context menu. The element may also be hidden from the Map Elements view.
  • Edit a map element: Right-click on the element and select Edit from the context menu. The element may also be edited from the Map Elements view.
  • Rename a map element: Right-click on the element and select Rename from the context menu. The element may also be renamed from the Map Elements view.
  • Move a map element: Place the cursor over the map element, press the left mouse button down and drag.

Profile
  • Adjust the length of the profile line: Place the cursor over one of the profile paddles, press the left mouse button down and drag.
  • Find the value of a point on the profile graph: Single-click with the left mouse button at the desired location in the graph view.
  • Open a closed profile graph: Open the Map Elements view and select the profile from the list. Press Open Graph.

Tape Measure
  • Measure the distance, slope, or height between two points: Place DERT in measuring mode by selecting the tape measure button. Single-click at the location of the first point with the left mouse button, then single-click at the location of the second point.  Read the values from the measuring tape window.

Scene Attributes
  • Change the color of the sky: Select Edit>Change Background Color to display the background dialog.  Enter red, green, blue and alpha values (0.0 - 1.0), select a predefined color, or use the color chooser.  Press OK.
  • Change to stereo viewing: Select Edit>Stereo. Select the stereo check box. Press OK.
  • Turn off the center of rotation cross hair: Select Edit>Landscape>Show Center of Rotation Cross hair.

Lighting and Shadows
  • Switch between main light modes: Open the lighting and shadows view. Press Solar or Artificial.
  • Enable/disable the headlight: Open the lighting and shadows view. Press Enable in the headlight section.
  • Enable/disable shadows: Open the lighting and shadows view. Press Enable Shadows. Use the light positioning view to change shadows.
  • Set the center and radius of the shadow sphere (area covered by shadows): Open the lighting and shadows view and enter the values (Note: You can copy and paste from the marble location field or other window).
  • Change the position of the light: Open the light positioning view. Adjust the position via the calendar or the az/el controls.
  • Change light properties: Open the lighting and shadows view and enter the values.

Surface
  • Change the vertical exaggeration of elevation: Open the surface and layers view. Change the Vertical Exaggeration field.
  • Change the terrain to display as wireframe: Open the surface and layers view. Select Display as wireframe.
  • Show the surface normals for the terrain: Open the surface and layers view. Select Show surface normals.
  • Change the surface color of the terrain: Open the surface and layers view. Press the Surface Color button.
  • Display a grid draped on the terrain surface: Open the surface and layers view. Select Surface Grid. Set Cell Size and Color.

Layers
  • Turn off the orthoimage layers: Open the surface and view. Deselect Show in the Layers section.
  • Display an image layer: Open the surface and layers view. Press Change. Select the destination layer in the left list. Select the layer from the right list and move it to the left list using the arrows. Press OK.
  • Display an elevation, slope, or aspect map: Open the surface and layers view. Press Change. Select the destination layer in the left list. Select the layer from the right list and move it to the left list using the left arrow. Press OK.
  • Set a color map for an elevation, slope, or aspect map: Open the color bars view and press Color Map for the layer.
  • Change the contribution of each layer: Open the surface and layers view.  Change the opacity spinner next to a layer to set the percentage.  All layers will adjust to sum up to 1. Deselect Auto-adjust Blend Factor to stop auto-adjustment.
  • Add a color map to DERT: add general color maps to dert/colormap in the installation directory, add landscape-specific color maps to the dert/colormap in the landscape.  Color map file format is described in the GDAL gdaldem documentation.

Camera
  • Add a camera definition: Copy a properties file in dert/camera to a new name and edit it with new camera parameters. A definition file may also be added to dert/camera in a landscape.
  • Open a closed camera view: Open the map elements view and select the camera from the list. Press Open View.
